# Java Console Calculator

A simple Java-based calculator that runs in the console and performs basic mathematical operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero check)
- Power (x^y)
- Square Root (of a single number)

## Requirements
- Java JDK 8 or later
- Terminal/Command Prompt

## How to Run
1. Save the code as `ConsoleCalculator.java`.
2. Open a terminal/command prompt in the file directory.
3. Compile the code: javac ConsoleCalculator.java
4. Run the program: java ConsoleCalculator

##Usage

-Follow the on-screen menu to select an operation.
-Enter the required numbers.
-To exit, choose the Exit option in the menu.
